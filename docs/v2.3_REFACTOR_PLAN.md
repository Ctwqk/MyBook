## v2.3 核心特性

### 1. Writer 分 scene 生成
```python
# 场景模式
writer_request = WriterGenerationRequest(
    chapter_id=1,
    use_scene_mode=True,
    scene_count=3,  # 3 scenes
    target_word_count=3000
)
```

### 2. 三种运行模式
```python
OperationMode.BLACKBOX      # 完全黑箱，无需人工
OperationMode.CHECKPOINT    # 检查点模式，关键节点等待确认
OperationMode.COLLABORATIVE # 共驾编辑模式
```

### 3. 重试策略
```python
RetryPolicy(
    max_retries=2,
    max_repairs=1,
    writer_retry_config={...},
    reviewer_retry_config={...},
)
```

### 4. 多项目隔离
```python
# 每个操作都验证 project_id
if chapter.project_id != project_id:
    raise ValueError("Project ID mismatch")
```
